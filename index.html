Smart Start

In het kader van Smart Start heb ik geleerd te analyseren, communiceren en mijzelf professioneel te ontwikkelen. Dit heb ik op verschillende wijzen gedaan. Onderstaand kun je lezen hoe ik dit heb aangepakt.

Herkennen, analyseren en diagnosticeren
Tijdens de Minor heb ik geleerd hoe ik een Smart Industry bedrijf kan herkennen. Het beste kun je dit doen met onderstaande afbeelding. Hier staan alle trends en ontwikkelingen duidelijk weergegeven.


Zelf heb ik een project gedaan bij Het Internet Huis. Het Internet Huis valt met name onder Smart Products en Network Centric. Het is een Connectiviteit R&D bedrijf dat op aanvraag van een klant een Smart Connection/IoT oplossing bedenkt. In het project heb ik samen met Sjors Broenland en Linda Jansen een Proof of Concept gemaakt van een IoT oplossing om senioren en mantelzorgers te helpen. Het project was daarmee een Smart Industry project.

Smart Technology

Tijdens de minor heb ik veel geleerd over technologieën die je kan toepassen om iets efficiënter te doen. Zo heb ik een ESP32 (kleine computer) geprogrammeerd, met sensoren gewerkt en zelf prototypes ontworpen. Hier onder meer daar over.

    Innoveren en ontwerpen
    Met Smart Technology heb ik geleerd op een innovatieve manier een prototype te maken. Zo heb ik een SMART plant gemaakt. Dit heb ik gedaan door gebruik te maken van de volgende zaken.
  • Een goedkope computer (ESP32);
  • Lichtsensor & vochtsensor;
  • Stroomvoorziening;
  • Een waterpomp;
  • Een app;
  • Een server;
  • Een SMART Socket;
  • PCB;
  • 3D- geprinte behuizing;
  • Een script.
Met deze middelen heb ik een product gemaakt waardoor de plant automatisch genoeg licht en water krijgt. Hierdoor heb je er geen omkijken meer naar. Dit is erg handig wanneer je op vakantie gaat of een efficiënte bedrijfsvoering van bijvoorbeeld een kas wilt voeren.

Naast het maken van een SMART plant ben ik naar de TU in Delft geweest om een robot te programmeren. Samen met andere studenten en docenten hebben we onder leiding van Tim Geers gezorgd dat een robotarm samen met een camera dobbelstenen kan herkennen, oppakken en verplaatsen.

Met dank aan Daniel Krois voor het maken van de video.

Onderzoeken
Om een SMART plant te maken heb ik hulp gehad van een guide, een expert en een community. Door de guide kwam ik door een deel van het proces heen. Wanneer ik ergens zelf niet uitkwam kon ik hulp van een expert of de community inschakelen. Ik heb gemerkt dat je veel sneller bij een eindresultaat komt wanneer je als groep iets probeert te bereiken. Zelf kun je ergens lang op vastlopen. Dan is het fijn dat je elkaar kunt helpen. Zelf heb ik hier ook een rol in gespeeld. Verder heb ik soms op websites gekeken waarop gepost wordt over problemen die ik tegenkwam. Mijn onderzoek bestond met name uit makkelijk toegankelijke, niet wetenschappelijke, bronnen. Ik heb ook bijgedragen aan het materiaal wat ik heb gebruikt voor onderzoek. Zo heb ik een fout in een guide aangekaart en ben ik met een oplossing voor het probleem gekomen. Verder heb ik actief samengewerkt en anderen in de community geholpen.

Communiceren
Door een fout in een guide te melden en deze helpen verbeteren communiceer ik naar de makers zodat ze een volgende groep een betere versie kunnen bieden. Hiermee werk ik in kleine mate mee aan het verbeteren van het onderwijs voor de volgende generatie.

Smart Connection

    Ik heb via Informatica Actief de basis geleerd van
  • Digitale informatie;
  • Communicatie en taal van computers;
  • Computers en randapparatuur;
  • Besturingssystemen;
  • Computernetwerken;
  • Algoritmes;
  • Relationele databases;
  • DBMS;
  • Informatiemodellering.
Daarnaast weet ik nu wat het oplevert om via een connectie data aan te leveren en daar informatie uit te halen. Verwerken van data of een connectie maken kan op verschillende manieren, enkele zijn: R (codetaal), Python, blockchain of een IoT schakeling.

Innoveren en ontwerpen
Als vervolg vanuit mijn vorige cluster heb ik met behulp van R data uit een tekstbestand kunnen omzetten naar een visuele weergave ervan. Zo heb ik een Whatsapp gesprek gedownload en hier een wordcloud van gemaakt (niet op deze webpagina vanwege privacy). Ik heb geleerd dat het redelijk toegankelijk is om data visueel te maken. Zolang de data maar gestructureerd is kun je er alles mee doen. Daarbuiten heb ik geleerd om met ML5 zelf machine learning toe te passen. Op deze manier heb ik geleerd hoe ik een machine slimmer kan maken en zelf kan laten nadenken. Ik zie in dat machine learning een geweldige tool is en dat er zeker een rijke, maar ook verraderlijke, toekomst in ligt. Artificial Intelligence biedt geweldige mogelijkheden maar deze moeten wel gemonitord worden.

Onderzoeken
Om onderzoek te doen naar de tools die je kan gebruiken voor het verzamelen en verwerken van data heb ik verschillende manieren gebruikt. Om achter de tools te komen heb ik gebruik gemaakt van experts. Om daarna verder te leren over de tools heb ik een combinatie gebruikt van literatuuronderzoek en hulp van experts.

Programmeren
Ik had al ervaring met programmeertalen zoals HTML en Java maar nog niet met R of andere talen die veel gebruikt worden om data te verzamelen en/of te verwerken. Om dit te leren heb ik mij ingelezen en een guide doorlopen. Ook heb ik workshops gevolgd van experts. Blockchain
Ik was al redelijk bekend met blockchain doordat ik zelf deelneem in de cryptocurrency wereld. Ik lees bijvoorbeeld informatie op Reddit of kijk videos op Youtube. Zo leer ik elke dag een beetje bij over interessante technologische onderwerpen. De workshop blockchain tijdens de minor heeft mij ook erg geholpen om smart contracts beter te begrijpen. Het was erg interessant om zelf een smart contract te deployen op een testnet.

IoT
Om meer te leren over IoT heb ik met name gebruik gemaakt van online guides, experts en websites met informatie. Door deze factoren kun je zelf gemakkelijk een IoT schakeling maken.

Communiceren
Mijn wordcloud had ik gedeeld met de personen uit de groep waaruit de data kwam. Ze reageerden enthousiast en wilden weten hoe ik het gedaan had. Een iemand had ik op mijn laptop laten zien hoe ik de wordcloud gemaakt had. Diegene vond het erg interessant en heeft later ook zelf met R geëxperimenteerd.
